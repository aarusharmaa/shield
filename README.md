🛡️ Shield - Emergency Response System
A real-time disaster management platform that saves lives through instant alerts, safe navigation, and emergency communication.
🚨 Problem Statement
As disasters and wars increase globally, people face life-threatening situations daily. Traditional emergency systems are slow, fragmented, and often fail when needed most.
💡 Our Solution - SHIELD
Smart Hazard Intelligence and Emergency Life Defense
A comprehensive web-based platform that empowers users with:

⚡ Real-time disaster alerts
🗺️ Safe route navigation
📲 One-click emergency SMS system

🌟 Key Features
📡 Real-Time Disaster Alerts

🗺️ Smart Navigation & Safe Routes

Google Maps integration for live routing
Automatic hazard zone avoidance
Multiple evacuation route suggestions

📲 One-Click Emergency SMS

Twilio-powered instant messaging
Pre-configured emergency contacts
Automated location sharing

🔧 Tech Stack

Frontend: HTML5, CSS3, JavaScript (Vanilla)
APIs:

USGS Earthquake API
OpenWeatherMap API
Twilio SMS API
Google Maps API


🔑 API Configuration
Required API Keys:

USGS Earthquake API: No key required (public)
OpenWeatherMap API: Get free key at openweathermap.org
Twilio SMS API: Get credentials at twilio.com
Google Maps API: Get key at developers.google.com
